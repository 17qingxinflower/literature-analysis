# 文献分析工具 🎓

**一款 AI 驱动的研究助手，可自动分析学术论文并生成结构化报告。**

[🇨🇳 中文版本](README_zh.md) | [🌐 English Version](README.md)

---

## ✨ 功能简介

上传一篇研究论文，几秒钟内即可获得全面的分析报告，包括：

- 📋 **基础信息** - 标题、作者、关键词、发表信息
- 🧠 **内容分析** - 研究背景、方法、实验、结果
- 💡 **核心结论** - 以您首选语言呈现的核心发现
- 🎨 **方法流程图提示词** - 专业图表生成提示词，用于 AI 图像生成
- 🖼️ **AI 生图功能** - 根据提示词自动生成流程图（支持多种图像模型）
- 📊 **源码信息** - GitHub 链接和实现详情（如可用）

---

## 🚀 使用方法

### 1. 配置 API Key
点击 **"配置"** 按钮，输入您从 [platform.iflow.cn](https://platform.iflow.cn) 获取的心流 API Key

### 2. 上传论文
上传您的研究论文（支持 PDF、DOCX 或 TXT 格式）

### 3. （可选）上传参考图片
如果您有方法流程图的参考图片，可以在此上传

### 4. 选择 AI 模型
从多种 AI 模型中选择：
- **DeepSeek V3.2** （推荐）
- 智谱 GLM 系列
- 通义千问系列
- Kimi 系列
- 更多...

### 5. 开始分析
点击 **"开始分析"**，等待 1-3 分钟

### 6. 查看和下载
查看您的结构化报告，并以 Markdown 格式下载

### 7. 生成流程图（可选）
在报告下方，您可以：
- **复制提示词** - 一键复制流程图提示词
- **选择图像模型** - 从多种 AI 图像模型中选择
- **生成流程图** - 直接生成专业流程图
- **下载图片** - 下载生成的流程图

支持的图像模型：
- **Gemini 2.5 Flash Image** (Nano Banana) - 谷歌快速高质量模型
- **Gemini 3.0 Pro Image** (Nano Banana Pro) - 谷歌最新预览版
- **Kling V1/V2 系列** - 可灵图像生成模型

### 8. 分析另一篇论文
点击 **"再次上传"** 分析更多论文（您的设置已保存）

## 💡 最佳实践建议

### 文献分析
1. **使用高质量 PDF** - 基于文本的 PDF 比扫描版效果更好
2. **包含方法图** - 上传参考图片以获得更好的流程图提示词
3. **选择合适模型** - 对于大多数论文，推荐使用 DeepSeek V3.2
4. **审阅报告** - 始终审阅 AI 生成的内容以确保准确性

### 流程图生成
1. **配置七牛云 API Key** - 在设置中配置七牛云 API Key
2. **选择图像模型** - 推荐使用 Gemini 2.5 Flash Image（快速高质量）
3. **复制提示词** - 可复制到豆包、Gemini 等外部工具生成
4. **直接生成** - 在网页中直接生成并下载流程图

## 🆘 需要帮助？

### 常见问题

**问：我从哪里获取 API Key？**
答：访问 [platform.iflow.cn](https://platform.iflow.cn) 获取您的免费 API Key

**问：七牛云 API Key 在哪里获取？**
答：访问 [七牛云 AI 大模型](https://portal.qiniu.com/ai/llm) 获取图像生成 API Key

**问：分析失败？**
答：检查您的 API Key 是否有效且余额充足

**问：文件上传失败？**
答：确保文件小于 50MB 且为支持的格式

**问：分析质量差？**
答：尝试不同的 AI 模型，或确保 PDF 基于文本（非扫描版）

**问：流程图生成失败？**
答：检查七牛云 API Key 配置，确保选择的模型支持文生图功能

**问：如何获得更好的流程图？**
答：推荐使用 Gemini 2.5 Flash Image 模型，或者复制提示词到豆包、Gemini 等外部工具生成
