# Flowchart Designer

This skill generates detailed, professional prompts for creating academic method flowcharts that can be used with AI image generation tools like Midjourney, DALL-E, or Gemini.

## Capabilities

1. **Professional Academic Style**: Generates prompts following IEEE/Nature/Science journal standards
2. **Detailed Structure**: Multi-stage layouts with clear data flow arrows
3. **Technical Accuracy**: Preserves all technical details, parameters, and equipment names
4. **Aesthetic Design**: Soft pastel colors, rounded rectangles, clean typography
5. **Chinese Text Support**: All annotations in Simplified Chinese

## When to Use

Invoke this skill when:
- User wants to generate a method flowchart for a research paper
- User asks for flowchart prompt generation
- User needs to visualize experimental procedures
- User wants to create professional academic diagrams

## Prompt Generation Guidelines

### 1. Overall Style Requirements
- High-resolution, professional academic methodological flowchart
- Follow top-tier IEEE journal paper aesthetic style
- Use rounded rectangular stages with soft pastel background colors
- Moderate aspect ratio (e.g., 16:9 landscape), not too wide or narrow
- Aesthetically pleasing color palette using soft pastels (blues, oranges/yellows, greens)

### 2. Structure Requirements
- Horizontally or vertically divided into main stages (rounded rectangles)
- Connected by thick, clear data flow arrows
- Each stage has clear title and sub-modules
- Complete flow with input, processing, and output

### 3. Content Requirements
- **All text annotations must be in Simplified Chinese**
- Accurately reflect technical details and parameters
- Include specific numerical values, equipment names, algorithm names
- Mark key technical points and innovations

### 4. Visual Elements
- Use icons for different component types (sensors, algorithms, databases)
- Use different colors to distinguish functional modules
- Add necessary annotations and explanation boxes
- Ensure text is clear and readable

## Output Format

The skill generates prompts in the following format:

```
Prompt for Professional Scientific Illustration AI:

Create a high-resolution, professional academic methodological flowchart diagram...

[Detailed description including:
- Overall layout and style
- Specific content of each stage
- Arrows and data flow direction
- Color scheme
- Text annotations (Chinese)
- Technical details]
```

## Example Usage

**Input**: Experimental method description with steps, parameters, and equipment

**Output**: Professional flowchart prompt ready for AI image generation

## Integration

This skill integrates with the literature analysis system to automatically generate flowchart prompts from extracted experimental methods.
