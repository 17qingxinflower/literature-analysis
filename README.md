# Literature Analysis Tool 🎓

**An AI-powered research assistant that automatically analyzes academic papers and generates structured reports.**

[🇨🇳 中文版本](README_zh.md) | [🌐 English Version](README.md)

---

## ✨ What It Does

Upload a research paper and get a comprehensive analysis report in seconds, including:

- 📋 **Basic Information** - Title, authors, keywords, publication details
- 🧠 **Content Analysis** - Research background, methods, experiments, results
- 💡 **Key Conclusions** - Core findings in your preferred language
- 🎨 **Method Flowchart Prompt** - Professional diagram prompts for AI image generation
- 🖼️ **AI Image Generation** - Automatically generate flowcharts from prompts (multiple models supported)
- 📊 **Source Code Info** - GitHub links and implementation details (if available)

---

## 🚀 How to Use

### 1. Configure API Key
Click the **Settings** button and enter your iFlow API Key from [platform.iflow.cn](https://platform.iflow.cn)

### 2. Upload Paper
Upload your research paper (PDF, DOCX, or TXT format)

### 3. (Optional) Upload Reference Image
If you have a reference image for the method flowchart, upload it here

### 4. Select AI Model
Choose from multiple AI models:
- **DeepSeek V3.2** (Recommended)
- GLM-4 Series (Zhipu)
- Qwen Series (Alibaba)
- Kimi Series (Moonshot)
- And more...

### 5. Start Analysis
Click **Start Analysis** and wait 1-3 minutes

### 6. View & Download
Review your structured report and download it in Markdown format

### 7. Generate Flowchart (Optional)
Below the report, you can:
- **Copy Prompt** - One-click copy of the flowchart prompt
- **Select Image Model** - Choose from multiple AI image models
- **Generate Flowchart** - Directly generate professional flowchart
- **Download Image** - Download the generated flowchart

Supported image models:
- **Gemini 2.5 Flash Image** (Nano Banana) - Google's fast high-quality model
- **Gemini 3.0 Pro Image** (Nano Banana Pro) - Google's latest preview model
- **Kling V1/V2 Series** - Kling professional image generation models

### 8. Analyze Another Paper
Click **Upload Again** to analyze more papers (your settings are saved)

## 💡 Tips for Best Results

### Literature Analysis
1. **Use High-Quality PDFs** - Text-based PDFs work better than scanned documents
2. **Include Method Diagrams** - Upload reference images for better flowchart prompts
3. **Choose the Right Model** - DeepSeek V3.2 is recommended for most papers
4. **Review the Report** - Always review AI-generated content for accuracy

### Flowchart Generation
1. **Configure Qiniu API Key** - Set up Qiniu Cloud API Key in settings
2. **Select Image Model** - Gemini 2.5 Flash Image recommended (fast & high-quality)
3. **Copy Prompt** - Can copy to external tools like Doubao, Gemini for generation
4. **Generate Directly** - Generate and download flowcharts directly in the webpage

## 🆘 Need Help?

### Common Issues

**Q: Where do I get an API Key?**
A: Visit [platform.iflow.cn](https://platform.iflow.cn) to get your free API Key

**Q: Where do I get Qiniu Cloud API Key?**
A: Visit [Qiniu AI Large Models](https://portal.qiniu.com/ai/llm) to get image generation API Key

**Q: Analysis failed?**
A: Check your API Key is valid and has sufficient balance

**Q: File upload failed?**
A: Ensure your file is under 50MB and in a supported format

**Q: Poor analysis quality?**
A: Try a different AI model or ensure the PDF is text-based (not scanned)

**Q: Flowchart generation failed?**
A: Check Qiniu Cloud API Key configuration and ensure the selected model supports text-to-image

**Q: How to get better flowcharts?**
A: Recommend using Gemini 2.5 Flash Image model, or copy the prompt to external tools like Doubao, Gemini for generation
